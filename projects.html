Little Mads' Lily Pad

Little Mads' Lily Pad is a fun little browser game with a simple objective: Capture as many flies as you can before time runs out! The game was designed for the iPhone 6 and may not fit properly on all devices. It was made with the Pixi.js graphics library and JavaScript. This is the first game I created, and I can't wait to make many more.

Tic Tac Toe

I’m making Tic Tac Toe with the Raylib graphics library in C++. I have a few goals with this project: I want to become more comfortable consuming open-source libraries in C++ with CMake, I want to create an engine I can continue to use as a starting point for a future game I plan on making, and I want to develop a game that runs on multiple platforms (Desktop, Web, and Mobile). I’m still working on this game, but I have linked the source code for anyone who wishes to see the development.
